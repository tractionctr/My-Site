# Sticky Notes (Static Version)

This is a simplified static version of a Django sticky notes app, converted for GitHub Pages deployment.

## What it is
A basic sticky notes UI that allows users to:
- View sample notes
- Navigate to a simple form page
- Explore a frontend-only version of the original Django project

## Tech used
- HTML
- CSS

## Note
This is a static demo version. It does NOT include backend functionality (no database, no saving notes).

## Pages
- index.html → Notes list
- form.html → Create note page
